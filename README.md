°Como ejecutar el proyecto

|Clonar los archivos

|Conectar cada rama 

|Meterte a la rama dev (Para no subir ningun archivo por error a la main)

|Ejecutar el archivo "menu.py" y probar la aplicacion


°Como puedo clonar los archivos?
Tienes que primero copiar el link del repositorio que se te proporsiona (o ir al github) luego en el visual te metes en la opcion terminal y le das a new terminal esta te mandara directamente a una terminal de powershell y aqui es simplemente colocar el siguiente comando "$ git clone LINK DEL REPOSITORIO".
Si quieres que trabajarlo con git solo le tienes que dar al mas y darle click a git bash.
